Reports on any calls to .toString() used in string concatenations and as arguments to the print and println methods of java.io.PrintWriter and java.io.PrintStream. The conversion to string will be handled by the underlying library methods without an explicit call to .toString().

Note that without the .toString() the expression will have different semantics (the string "null" will be used instead of throwing a NullPointerException).
